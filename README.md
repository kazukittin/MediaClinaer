# MediaClinaer

ブレ画像などの低品質画像を検出し、確認しながら安全に整理するためのWindows向け画像整理ツールです。

## 開発方針

このプロジェクトはウォーターフォール開発で進めます。

1. 要件定義
2. 基本設計
3. 詳細設計
4. 実装
5. テスト
6. リリース準備

各工程の成果物は `docs/` に残し、合意済みの仕様を基準に次工程へ進みます。

## 確定した初期方針

- 対象OSはWindowsのみ
- Python + PySide6 のデスクトップアプリとして開発
- Portable配布前提で開発
- 初期版では完全削除を行わず、隔離フォルダへの移動のみ
- 類似画像、類似動画、完全重複検出は対象外
- NASはWindowsから参照できるネットワーク共有フォルダとして対応
- 隔離フォルダはPCローカル上の1か所にまとめる
- NAS上のファイルもPCローカルの隔離フォルダへ移動する
- 検出結果と解析結果はキャッシュとして保存する方針
- キャッシュは無期限に保持する
- 隔離フォルダとキャッシュフォルダはアプリフォルダ配下に作成し、完全パスで紐づける
- 対応画像形式にJPGを含める
- 長いパスは可能な限り扱い、処理できないファイルはスキップしてログに記録する
- 検出結果、解析キャッシュ、隔離履歴はSQLiteに保存する
- 設定ファイルはJSONに保存する

## 初期スコープ

- ブレ画像の検出
- ローカルフォルダとNAS共有フォルダのスキャン
- 削除前の確認画面
- 削除ではなく隔離フォルダへの移動
- 前回スキャン結果の再表示
- 解析済みファイルのキャッシュ利用
- 隔離ファイルと元ファイルの完全パス管理

## 安全方針

初期版ではファイルを即時完全削除しません。誤削除を避けるため、整理対象は隔離フォルダへ移動し、ユーザーが後から戻せる設計にします。

## 配布方針

インストーラーを使わないPortable配布を前提にします。アプリ本体のあるフォルダ配下に `cache`、`logs`、`quarantine` を作成し、データをまとめて持ち運べる構成にします。

## 開発環境での起動

```powershell
.venv\Scripts\python.exe -m media_clinaer.main
```

## テスト

```powershell
.venv\Scripts\python.exe -m pytest
```
